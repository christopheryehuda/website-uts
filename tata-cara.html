2. Tata Cara Membuat Website Sederhana

  1. Buka editor seperti VSCode atau Notepad++.
  2. Buat file index.html dan tulis struktur dasar HTML.
  3. Tambahkan gaya menggunakan CSS atau framework seperti Tailwind.
  4. Gunakan JavaScript untuk menambahkan interaktivitas.
  5. Simpan semua file dalam satu folder dan buka di browser untuk melihat hasilnya.
